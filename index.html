About Me

I am a Gameplay Programmer focused on building gameplay systems in Unreal Engine using C++ and Blueprint.

My work centers around designing modular and scalable systems for interaction, combat, AI, and multiplayer gameplay, with an emphasis on reusable and extensible architectures that support different gameplay scenarios.

This portfolio showcases selected systems and prototypes that reflect my approach to gameplay programming and system design.









Portfolio

Dice System Plugin

A modular, reusable dice mechanic plugin developed in C++ and Blueprint for Unreal Engine 5. Designed for turn-based gameplay, it features drag-and-drop dice slotting, outcome resolution driven by data tables, and a demo level showcasing player interaction. The system is lightweight and flexible, making it well suited for prototyping RPG and board-game-inspired mechanics.

Key Features:

  • Multi-dice rolling system with Fail, Normal, and Strong outcomes
  • Outcome resolution powered by a customizable DataTable
  • Drag-and-drop dice slot UI with Blueprint integration
  • Clean separation of logic and UI via controller
  • Includes a working demo level with a player character and an interactable target (no AI or combat)

Wavefront — 3D Tower Defense / Action Hybrid Prototype

Wavefront is a Unity 6 prototype that merges classic Tower Defense dynamics with direct player control and a modern modular architecture. The design prioritizes performance, scalability, and maintainability.

  • Modular Architecture: Independent managers and a central Service Provider for cross-cutting systems
  • Performance: Object Pooling for efficient enemy spawning/destruction
  • Data-Driven Design: Scriptable Objects / serialized configs for iteration
  • Composition: Character features built via components over inheritance
  • AI: NavMesh Agents
  • Combat: Auto target selection and range-based fire triggers
  • Wave Logic: LinkedList-driven progression and timer-based wave classes
  • UI: MVP (Model–View–Presenter) pattern
