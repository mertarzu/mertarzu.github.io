Dice System Plugin

A modular, reusable dice mechanic plugin developed in C++ and Blueprint for Unreal Engine 5. The system is lightweight and flexible, making it well suited for prototyping RPG and board-game-inspired mechanics.

Key Features:

  • Multi-dice rolling system with Fail, Normal, and Strong outcomes
  • Outcome resolution powered by a customizable DataTable
  • Drag-and-drop dice slot UI with Blueprint integration
  • Clean separation of logic and UI via controller
  • Includes a working demo level with a player character and an interactable target (no AI or combat)
