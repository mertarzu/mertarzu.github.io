As a solo developer, I'm committed to leveraging UE5's capabilities to create an imaginative and engaging gaming experience.

High Mountain

High Mountain is my first Unreal Engine 5 learning project, in which I explored and implemented a variety of core gameplay mechanics and engine features. I developed this third-person action RPG prototype to gain hands-on experience with:

  • Motion Warping
  • MetaSounds
  • Linked Anim Graphs
  • UE5's New Retargeting System (with IK Rigs and IK Retargeters)
  • Leg IK with Control Rigs
  • Visual Effect Creation in Niagara
  • Enhanced Input System
  • UMG (Unreal Motion Graphics)
  • Static Lighting and Lightmass Importance Volume
  • Post-Process Effects

This project solidified my understanding of Unreal Engine 5's capabilities and provided valuable insights into the game development process.

MTPShooter

MTPShooter is my first entirely blueprint project, a multiplayer third-person shooter prototype. This project was a foundational learning experience in developing core game mechanics and implementing multiplayer functionality within Unreal Engine 5. I utilized and retargeted animations from the Animation Starter package to UE5 and created a custom idle animation suitable for a shooter. Grenade-throwing animation was sourced from the Lyra example project, and weapons were integrated from the Military Weapons Silver Asset Pack. Visual effects were enhanced using assets from the Realistic Starter VFX Pack. I adapted Unreal Engine's Online Subsystem Sessions, based on the Multiplayer Shootout Sample, to enable online multiplayer gameplay. Two core gameplay abilities were implemented: shooting, featuring a dynamic crosshair for accurate aiming, and grenade throwing, which includes a spline-based path prediction system for precise grenade trajectory visualization. All gameplay elements, including procedural aim offset, particle effects, sounds, blood spatter, health, death, and respawning, are fully replicated for a seamless multiplayer experience. A replicated overhead health bar shows the player's health status.
